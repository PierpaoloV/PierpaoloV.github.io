# Context

## Domain Modules

- **Research Area**: A reusable line of work that can be projected onto the homepage, CV, projects, and publications.
- **Project Catalog**: The owning module for public project records that appear in the Projects page and the CV.
- **Publication Catalog**: The owning module for publication records that appear in the Publications page.
- **Skills Inventory**: The owning module for the portfolio's technology and capability vocabulary, rendered as featured homepage groups and CV tracks.
- **Learning Log**: The owning module for the monthly learning schedule plus the runtime rules that derive progress from the external repository.
- **Theme State**: The owning module for the site's light/dark mode state and theme-token propagation.

## Current Adapters

- The homepage is a projection of the profile, research areas, and featured skill groups.
- The CV is a projection of resume links, career records, education records, the Skills Inventory, and selected Project Catalog entries.
- The Projects page is a projection of the Project Catalog.
- The Publications page is a projection of the Publication Catalog.
- The Learning page is a projection of the Learning Log.
